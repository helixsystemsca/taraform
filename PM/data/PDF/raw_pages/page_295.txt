Project dashboards. A project dashboard is a set of charts and graphs showing progress or
performance against important measures of the project. A common way of showing large quantities
of information on metrics is a dashboard. Dashboards generally collect information electronically and
generate charts that depict status. Often, dashboards offer high-level summaries of data and allow
drill-down analysis into contributing data. Figure 5-16 and Figure 5-17 each provide examples of a
dashboard.
Dashboards often include information displayed as stoplight charts (also known as red–amber–green
[RAG] charts), bar charts, pie charts, and control charts. A text explanation can be used for any
measures that are outside the established thresholds.
